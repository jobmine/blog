C++ 标准模板库 (STL) 之一二

  • 1979年前后，C++ 诞生。
  • 1992年，C++ 模版诞生。
  • 1994年，HP实验室的 Stepanov 和 Meng Lee 完成了标准模板库的开发，并根据两个人的名字命名为：[ST]epanov [L]ee，译为 Standard Template Library。
  • 1995年，以80%的投票通过率，在 Waterloo 举办的 ANSI/ISO C++标准委员会成为 C++ 标准的一部分。

他们所包含的操作：

Sequence Container 有续容器

有序容器包含 vector, list 以及 deque，这里我们只讨论 vector, list。

vector 允许任意顺序的操作数据（借由 [] 或 at()），而 list 必须按顺序操作（借由迭代器）。所以 list 在表现上，删除、添加一个元素都是 O(n)。

vector Performance

行为 API 复杂度
随机访问 [], at() O(1)
删除添加最后项 push_back, pop_back O(1)
删除添加第一项 不支持API操作 O(N)
删除添加随机项 insert, erase O(N)

list Performance

行为 API 复杂度
随机访问 不支持API操作 O(N)
删除添加最后项 push_back, pop_back O(1)
删除添加第一项 push_front, pop_front O(1)
删除添加随机项 insert, erase O(N) to get there, O(1) to do

Iterator 迭代器

迭代器的功能很简单，我给一个位置 (i1)，然后做一些事情，然后去下一个位置，做一些事情，一直循环，直到我去了下一个位置，这个位置到达了另一个位置 (i2)。那么，如果一个函数需要调用 Iterator，你只需要准备好这个 i1 和 i2 就好了。例如：foo(i1, i2, ...)。

STL 的迭代器分为五个不同级别，最底权限的 input iterator 和 output iterator （使用 istream, ostream），forward iterator （unordered_set, unordered_map），bi-directional iterator (list, set)，random access iterator（vector, deque）。最高层的迭代器可以替代底层迭代器的功能，也就是说，vector 可以当做 iostream 来用。

  • bi-direction 一般使用 ++ 和 -- 实现 forward 和 backward 功能。
  • random access 一般需要 ++, -- 以及 [], +, -, +=, -= 实现任意访问目的。
